# CLI (optional) / 命令行工具（可选）

These scripts are dependency-free and meant for quick querying/validation.

## Query / 检索

- `python3 05_Resources/network/_cli/network.py <keywords>`
- `python3 05_Resources/network/_cli/network.py --tag '#resource/intro' <keywords> --json`
- `python3 05_Resources/network/_cli/network.py --needs-followup`

## Validate / 校验

- `python3 05_Resources/network/_cli/validate_people.py`
- `python3 05_Resources/network/_cli/validate_people.py --json`

On Windows, `python` may work better than `python3`.
