# Values / 价值观（Example）

> 说明：脱敏示例，仅用于展示结构与写法；请用你自己的版本替换。

## Core Values / 核心价值观

- 真实与可复盘：宁愿慢一点，也要能解释“为什么这样做”，并在未来复盘时能对齐当时的意图。
- 长期主义但不空想：偏好能积累的资产（技能/作品/论文/关系/信誉），拒绝只追短期情绪波动。
- 协作的尊重：把协作视为长期关系；少“情绪化甩锅”，多“结构化对齐”。
- Gentle Rationality：对重要之物愿意建立系统，但系统不绑架生活。

## Non-Goals / 非目标

- 不把“看起来很厉害/很忙”当成果。
- 不追求把生活完全数据化（指标只服务决策，不服务自我审判）。
- 不做“什么都收”的笔记堆积；宁可删掉，也不留无法对象化的噪声。

## Red Lines / 红线（不可妥协）

- 不入库可直接滥用的明文机密（密码/API key/私钥等）。
- 不为了短期收益牺牲信誉（夸大成果、引用不实、数据不透明）。
- 不长期处在“高消耗低回报”的环境（持续加班但没有成长/产出/尊重）。

## What This Constrains / 它约束什么

- 项目选择：优先做“可复现 + 可交付 + 可公开”的项目；避免只靠叙事的项目。
- 合作方式：优先与能明确预期、尊重边界、愿意对齐目标的人合作。
- 工具偏好：默认 search-first + repo-first；脚本只是加速器。

- 触发复查：遇到重大决策（换城市/换方向/换组织）或出现长期不适（连续 2–4 周）。
