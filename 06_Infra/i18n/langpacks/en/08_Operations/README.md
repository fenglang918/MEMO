# Operations

This directory holds the "system in motion" layer instead of goals themselves.

## Entry Points

- `00_Master-Schedule/`: cross-project schedule hub
- `01_Time-Tracking/`: time logging and lightweight reports
- `Mac-Setup/`: environment setup checklist

## Recommended Reading Order

1. `00_Master-Schedule/README.md`
2. `01_Time-Tracking/README.md`
3. `Mac-Setup/README.md`

## Built-in Examples

The template ships with sanitized sample files under `00_Master-Schedule/` and `01_Time-Tracking/`.
You can remove them with:

```bash
python3 06_Infra/cleanup_examples.py
```
