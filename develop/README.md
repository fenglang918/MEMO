# develop

这个目录是模板自身的开发工作区，用于放：

- 架构说明
- 变更计划
- 回流 / 重构记录
- 辅助开发产物

约定：

- 计划文档统一放在 `develop/plans/`
- 这里服务的是“模板如何演进”，不是个人项目本体内容

## 当前入口

- `docs/README.md`
  - 模板文档总入口
- `docs/overview/`
  - 模板的一页式架构、features、skills 总览
- `plans/README.md`
  - 模板重构、升级与回流计划入口

## 边界

- 这里只讲 `memo-publishing/pre-publish/repo/` 如何演进。
- 私有真源仓的内部说明稿、读者定制材料、真实讨论记录，不直接搬进这里。
