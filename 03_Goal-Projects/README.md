# Projects (Target / Plan / Cognition / Decision)

English:
- A “project” is a folder that contains: Target, Plan, Cognition notes, and Decisions.
- This keeps execution, thinking, and trade-offs close to each other.

中文：
- 一个“项目”是一个目录：Target（目标）、Plan（计划）、Cognition（认知笔记）、Decision（决策记录）。
- 好处是：执行/思考/取舍贴在一起，便于复盘与检索。

## Template / 模板

Copy `03_Goal-Projects/active/_TEMPLATE/` to create a new project folder.

## Examples / 示例（脱敏）

- 开源模板化（TPCD 示例）：`03_Goal-Projects/active/2026-01-MEMO-开源模板发布/`
- 高机会成本决策（TPCD 示例）：`03_Goal-Projects/active/2026-01-读博-vs-直接工作/`
