# Fields and tags (reference)

## Minimum fields (for retrieval + action)

- `Handle/ID`
- `关键词`
- `Tags`
- `最后联系` (recommended)
- `下次跟进` (recommended; or mark `暂无`)

## Tag prefixes (recommended)

- `#domain/...`
- `#resource/...`
- `#role/...`
- `#city/...`
- `#rel/...`
- `#status/...`

## Status

- `#status/active`: ongoing, high-signal
- `#status/warmup`: low-frequency, light-touch
- `#status/dormant`: keep, but no near-term action
- `#status/archived`: intentionally cold/closed
