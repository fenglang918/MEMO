# Workflow Checklist

## A. Intake

- [ ] Confirm source path is under `01_Inbox/`
- [ ] Read protocol files before routing
- [ ] Capture destination recommendation + one alternative

## B. Archive Write

- [ ] Create destination folder with normalized name
- [ ] Move source files into destination as `raw-*`
- [ ] Create `README.md` structured summary
- [ ] Create one `*.v1.json`
- [ ] Update one upstream index backlink

## C. Validation

- [ ] Source no longer stranded in Inbox (unless copy-mode requested)
- [ ] `README.md` exists
- [ ] `*.v1.json` exists and is parseable JSON
- [ ] All `raw-*` files exist with expected count
- [ ] Relative links are reachable
- [ ] Final report includes routing reason and rename map
