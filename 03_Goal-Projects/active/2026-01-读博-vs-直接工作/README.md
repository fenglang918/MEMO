# 读博 vs 直接工作（Target）

> 说明：脱敏示例项目。用于演示一个“高机会成本决策”如何落到 TPCD；更敏感的个人细节与部分讨论已从公开模板版移除。

## 目标（What）

澄清“读博”和“直接工作”的本质区别，并据此做出可复盘的路径选择。

## 为什么（Why）

- 这是一个高机会成本决策；不把“本质区别/评价函数/反馈回路”想清楚，容易用错误维度做选择。
- 该决策会影响未来 3–7 年的时间分配、能力结构、社交网络与可选机会集。

## 关键结果（KRs）

1. 形成一份可复用的对比框架与关键维度列表：`cognition/001-phd-vs-job-essence.md`
2. 完成信息收集与现实检验（访谈/试做/对齐约束）并记录结论：见 `PLAN.md`
3. 在明确的截止日期前做出选择，并记录理由与后续行动：新增 `decisions/001-choose-path.md`（待建）
