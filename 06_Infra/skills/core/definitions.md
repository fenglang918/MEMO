# Definitions（核心术语）

- **Ephemeral**：短暂碎片输入（聊天/截图/灵感/感受），默认进入 `01_Inbox/`
- **Object**：可长期维护的结构化对象（可版本化、可检索、可演化）
- **Assets / ME**：你“拥有”的稳定存量（能力、作品、偏好、身份锚点），放在 `04_Assets/`
- **Resources / MO**：你“可调用”的外部杠杆（人、工具、资源），放在 `05_Resources/`
- **Projects**：行动与推进的载体（Target/Plan/Cognition/Decision），放在 `03_Goal-Projects/`
- **Desires**：上游驱动层（我想要/我在乎/不想要），放在 `02_Desires/`
