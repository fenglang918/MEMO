# Inbox / 临时收集

English:
- Capture first, decide later.
- Review weekly: archive, promote into `02_Desires/` / `05_Resources/` / `04_Assets/` / `03_Goal-Projects/`, or delete.

中文：
- 先收集，后决策。
- 建议每周清理：归档、迁移到 `02_Desires/` / `05_Resources/` / `04_Assets/` / `03_Goal-Projects/`，或直接删除。

## AI Native（重点）

在 MEMO 的工作方式里，你不一定需要手工把“截图/聊天记录”先落盘到 `01_Inbox/raw/`。

更常见的流程是：**直接把截图/对话上下文交给 Code Agent**，让它端到端完成 PRISM 折射与对象化，然后只把结构化结果写入仓库（例如人脉卡片、Me 的 signals）。

当你确实想保留“原始材料”作为证据/回溯时，再把文件放进 `01_Inbox/raw/`（并在处理完成后清理或迁移）。
