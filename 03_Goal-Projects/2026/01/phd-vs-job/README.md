# 学术研究路径 vs 直接工作（Target）

> 说明：这是公开模板中的脱敏示例项目。
> 它用来演示一个“高机会成本路径选择”如何落到 TPCD，不代表任何真实个人的最终去向。

## 目标（What）

澄清“学术研究路径”和“直接工作路径”的本质区别，并在明确时间窗口内做出可复盘的选择。

## 为什么（Why）

- 这类路径选择通常会影响未来 3–7 年的时间分配、能力资本、社交网络与机会集。
- 如果不先澄清目标函数、反馈回路与约束条件，决策很容易被短期情绪、身份想象或外部噪声劫持。
- 这也是一个适合用 MEMO 记录的典型问题：有长期影响、证据分散、需要多轮更新与回看。

## 关键结果（KRs）

1. 形成一份可复用的对比框架与评价维度列表：
   `cognition/001-phd-vs-job-essence.md`
2. 完成现实检验：
   访谈、试做、约束梳理与反事实检查，见 `PLAN.md`
3. 在明确的决策截止日期前写出 ADR：
   `decisions/001-choose-path.md`
