# Role Models

这里放“我想成为的人”的参照系。

这层是 `02_Desires/` 的可选进阶骨架，用来回答：

- 我长期想靠近哪种人
- 我想学的是哪些能力、气质、选择方式
- 这些驱动如何回流到具体项目与计划

## 推荐工作流

- 新建人物条目：从 `people/_template.md` 复制到 `people/<handle>.md`
- 主题归纳（可选）：把多个 role model 的共性抽到 `themes/`
- 闭环：
  - 在 `03_Goal-Projects/` 的项目文档里链接相关 role model
  - 在 role model 条目里写 `related_goals`

## 边界

- 如果某个人是现实可触达、可合作、可跟进的人，更适合同时在 `05_Resources/network/` 建人卡
- 这里更关注“我想学什么、为什么想学”，而不是现实联络动作
