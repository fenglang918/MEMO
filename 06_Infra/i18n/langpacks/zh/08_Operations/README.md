# Operations

这里放“系统在运行中”的内容，而不是目标本身。

## 模块入口

- `00_Master-Schedule/`：跨项目总日程中枢
- `01_Time-Tracking/`：时间投入记录与轻量报表
- `Mac-Setup/`：环境配置 checklist

## 推荐阅读顺序

1. `00_Master-Schedule/README.md`
2. `01_Time-Tracking/README.md`
3. `Mac-Setup/README.md`

python3 06_Infra/cleanup_examples.py
